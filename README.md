# ✈️ CareerPilot

**CareerPilot** is a full-stack job search and recruitment management platform designed for three types of users — job seekers, companies, and admins. Job seekers can browse listings, track applications, and build ATS-optimized resumes. Companies can post and manage job listings and review applicants. Super admins oversee the entire platform.

🔗 **Live Site:** [https://career-pilot-nu.vercel.app](https://career-pilot-nu.vercel.app)

## 👥 Roles & Features

### 👤 User (Job Seeker)

- Register and log in securely
- Browse all available job listings
- Filter jobs by **Position** — Frontend, Backend, Fullstack, Internship
- Filter jobs by **Type** — Full-time, Part-time, On-site, Remote
- Apply to jobs and track application status
- View a personal dashboard showing:
  - Total applications submitted
  - Jobs currently in the interview stage
  - Rejected applications
- Build an **ATS-optimized resume** using the AI-powered resume builder
- Analyze job search performance with built-in analytics

### 🏢 Company

- Post new job listings with details like position type, location, and requirements
- Edit or update existing job postings
- View and manage applicants who have applied to their listings
- Manage the full recruitment pipeline from one dashboard

### 🛡️ Super Admin

- Full oversight of the entire platform
- Manage all users and companies
- Monitor platform-wide activity and data

---

## 🧭 Core Features

### 📋 Smart Application Tracker

Track every job application in one place. The Kanban-style dashboard organizes your applications into stages — Applied, Interviewing, and Rejected — so you always know where you stand.

### 📄 ATS Resume Builder

Create polished, AI-enhanced resumes that are optimized to pass Applicant Tracking Systems (ATS) and get your profile in front of real recruiters.

### 📊 Insightful Analytics

Visualize your job search performance with detailed stats. Understand your success rate, track activity over time, and identify what strategies are working.

### 🔍 Unified Job Search

Search and filter job listings instantly. Filter by role type (Frontend, Backend, Fullstack, Internship) and work style (Remote, On-site, Full-time, Part-time) to find the right fit quickly.
